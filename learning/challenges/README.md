# Desafios de transferência

- `transfer-array-window`: combinar janela e frequência sem pista.
- `transfer-tree-levels`: resolver a mesma leitura de árvore com DFS e BFS.
- `transfer-dp-choice`: modelar estado e comparar com uma hipótese gulosa.

O enunciado específico pode ser trocado por um problema ainda não visto; a evidência só conta sem pista.
